# Términos y Condiciones (Demo)

Esta es una aplicación demostrativa sin garantías. Al registrarte aceptas:

1. Que los datos (incluyendo contraseña) se guardan en texto plano en `localStorage`.
2. Que esta demo no envía mails ni valida identidad.
3. Que puedes borrar tus datos limpiando el almacenamiento del navegador.
4. Que las promociones no tienen validez legal.

No usar para producción. Mejora: cifrado, backend seguro, política de privacidad.
